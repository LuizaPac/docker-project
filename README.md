# docker-project

Autores: Luiza Costa Pacheco, Julia Paoli de Andrade, Tiago Vieira Santos

Projeto feito para a matéria de Redes de Computadores I, ministrada na Universidade Federal do Rio de Janeiro no ano de 2025. Essa página tem como objetivo ser uma introdução aos conhecimentos necessários para começar o uso de docker, fazendo ligações com o conteúdo da matéria.
